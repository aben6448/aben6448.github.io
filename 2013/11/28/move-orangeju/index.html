折腾搬家

前几天刚建好没多久的橘子居，让爱折腾的我又搬家了

  • github page真是个好东西，免费，不限流量，虽然只能是静态的，但对个人博客来说，足够了
  • hexo也是个很好的东东，基于nodejs，支持插件和主题，速度超快，支持Markdown，写东西超爽
  • Mou真是写Markdown的利器，橘子居里留了一篇Mou的示例文档，可以随时查一下

现在就是这么个组合，自己还是很满意的
